## 📝 Sobre o Projeto
Este repositório contém a resolução dos desafios de lógica e programação propostos pelo **Curso em Vídeo (Mundos 1, 2 e 3)**. Este trabalho faz parte da minha jornada de aprendizado contínuo em **Análise e Desenvolvimento de Sistemas (ADS)** na **Estácio**.

O objetivo é consolidar fundamentos de programação, algoritmos e estruturas de dados utilizando a linguagem Python.

## 🚀 Tecnologias e Ferramentas
* **Linguagem:** Python 3.
* **IDE:** PyCharm (Configurado em máquina de alta performance com 128GB RAM).
* **Controle de Versão:** Git & GitHub.

## 📂 Estrutura do Repositório
* `Mundo 1`: Fundamentos (Tipos primitivos, operadores, módulos).
* `Mundo 2`: Estruturas de controle (Condicionais `if/else`, repetições `for/while`).
* `Mundo 3`: Estruturas compostas (Listas, Tuplas, Dicionários, Funções e Módulos).
